# ESPHome EUC - Подключение моноколеса к умному дому

Проект позволяет интегрировать электротранспорт (моноколесо) в умный дом с помощью ESP32 и ESPHome.
Можно получать телеметрию, отображать показатели на веб-интерфейсе и использовать данные в Home Assistant для автоматизаций.

## Возможности

- Подключение к моноколесу через Bluetooth Low Energy (BLE)
- Получение основных метрик: скорость, заряд, температура, пробег
- Веб-интерфейс с показателями и статусом подключения
- Интеграция с Home Assistant через ESPHome API

## Требования

- Плата ESP32 (например, ESP32 DevKit v1)
- Установленный ESPHome
- BLE-адрес устройства

## Установка

### 1. Клонирование проекта

```bash
git clone git@github.com:vaninanton/esphome-euc.git esphome-euc
cd esphome-euc
```

### 2. Определение MAC-адреса устройства
Подключи ESP32 к компьютеру и прошей вспомогательную прошивку, чтобы узнать BLE-адрес моноколеса:

```bash
esphome run esphome-euc-ble-mac-scanner.yaml
```

Скопируй нужный адрес и запиши его для следующего шага.

### 3. Настройка secrets.yaml
Создай файл secrets.yaml из примера:

```bash
cp secrets.yaml.example secrets.yaml
```
Заполни его своими параметрами:
```yaml
wifi_ssid: "ТвояСетьWiFi"
wifi_password: "ПарольОтWiFi"
ha_encryption_key: "КлючИзHomeAssistant"
euc_veteran_mac: "XX:XX:XX:XX:XX:XX"
```

### 4. Прошивка основной программы

Подключи ESP32 к компьютеру и прошей основной конфиг:

```bash
esphome run esphome-euc-veteran.yaml
```
После прошивки устройство автоматически подключится к Wi-Fi и BLE-устройству.
